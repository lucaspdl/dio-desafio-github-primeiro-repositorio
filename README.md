## Desafio de Projeto sobre Git/Github

*Oque é o Markdown?* 

R: Markdown é uma linguagem de marcação, utilizada na maioria das vezes para o uso de anotações. Foi criado em 2004 por John Gruber, nascido na Filadélfia, Pensilvânia. Aaron Swartz e Gruber trabalharam em equipe para a criação do Markdown, que tinham como objetivo de permitir pessoas a escreverem usando um formato de texto simples fácil de ler e escrever, opcionalmente convertê-lo em XHTML.

*Comandos* : 

git status: Mostra o status dos arquivos dentro do repositório. (repositório que você estiver)
 
git add: sinaliza para o github que o arquivo precisa subir para a nuvem.

git commit -m “escreva o comentário”: cria um commit para o arquivo adicionado anteriormente.

git push origin main: envia todos os commits que estão localmente, para a nuvem.
